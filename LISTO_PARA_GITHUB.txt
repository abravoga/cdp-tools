================================================================================
 PASOS PARA SUBIR A GITHUB
================================================================================

PASO 1: Crear repositorio en GitHub
----------------------------------------
1. Ve a https://github.com
2. Click en "+" -> "New repository"
3. Nombre: "cdp-tools" (o el que prefieras)
4. Descripcion: "Sistema de analisis y prediccion CDP con ML"
5. Visibilidad: Public (recomendado) o Private
6. NO marques "Initialize with README" (ya lo tenemos)
7. Click "Create repository"


PASO 2: Conectar repositorio local
----------------------------------------
Ejecuta estos comandos (reemplaza TU-USUARIO con tu usuario de GitHub):

cd "C:\Users\abravoga\cdp-tools"

git remote add origin https://github.com/TU-USUARIO/cdp-tools.git

git branch -M main

git push -u origin main


PASO 3: Verificar en GitHub
----------------------------------------
1. Ve a https://github.com/TU-USUARIO/cdp-tools
2. Verifica que veas el README.md
3. Verifica que hay 42 archivos
4. Verifica que NO esta config.py


PASO 4 (OPCIONAL): Personalizar
----------------------------------------
En GitHub, en tu repositorio:

1. Agregar Topics/Tags:
   - Settings -> About -> Topics
   - Agregar: cdp, cloudera, elasticsearch, kibana, machine-learning

2. Agregar descripcion:
   - Click en el engranaje junto a "About"
   - Description: "Sistema de analisis y prediccion CDP con ML"
   - Website: Tu URL si tienes

3. Verifica git status
4. Busca el error en Google/Stack Overflow

Errores comunes:

- "Password authentication removed"
  -> Usa Personal Access Token en vez de password

- "Permission denied"
  -> Verifica que la URL del repo sea correcta

- "Remote already exists"
  -> git remote remove origin (y vuelve a agregar)
